# What's in my PC

> **"분명히 만들어 뒀는데, 어디에 저장했더라?"**

PC에 저장된 문서의 **내용까지 검색**할 수 있는 데스크탑 검색 도구입니다.
파일명을 몰라도, 문서 안에 있는 문구 하나로 찾을 수 있습니다.

인터넷 연결 없이 **내 PC 안에서만 동작**하며, 외부 서버로 데이터가 전송되지 않아 보안 걱정 없이 사용할 수 있습니다.

---

## 주요 기능

- **HWP, PDF, Word, Excel** 등 문서 내용 기반 검색
- 파일명을 몰라도 **문서 안의 문구**로 검색 가능
- 한 번 분석해두면 앱을 껐다 켜도 **검색 데이터 유지** (매번 재분석 불필요)
- 신규/변경 파일만 자동 업데이트 (동기화 버튼 한 번으로 반영)
- 삭제된 파일은 동기화 시 자동으로 DB에서 제거
- **완전 오프라인 동작** — 인터넷 연결 불필요, 보안 걱정 없음

## 빠른 시작 (설치 없이 바로 실행)

**Node.js 설치 불필요 — exe 파일 하나로 바로 실행됩니다.**

👉 **[최신 버전 다운로드 (Releases)](../../releases/latest)**

1. `Whats_in_my_PC.zip` 다운로드
2. **용량이 넉넉한 드라이브** (D:, E: 등)에 압축 해제
3. `What's in my PC.exe` 실행
4. `+ 폴더 추가` 버튼으로 검색할 폴더 등록
5. `신규/변경 문서만 동기화` 클릭 → 완료 메시지 확인
6. 검색창에 찾고 싶은 내용 입력!

> **참고**: 분석 데이터(DB)는 exe 옆 `data/` 폴더에 저장됩니다.  
> C드라이브 용량 부족이 걱정된다면 여유 있는 드라이브에 압축 해제하세요.

## 사용 팁

**☆ 첫 동기화는 여유 있을 때**  
문서 양에 따라 처음 분석에 1~2시간 소요될 수 있습니다.  
점심시간이나 퇴근 전에 실행해두는 것을 추천합니다.  
동기화 중에는 컴퓨터가 느려질 수 있으니 가급적 다른 작업은 잠시 쉬어주세요.

**☆ 두 번째부터는 빠릅니다**  
변경되거나 새로 추가된 파일만 분석하므로, 이후 동기화는 몇 초~몇 분이면 끝납니다.

**☆ 스캔한 PDF는 인식 불가**  
이미지로 스캔된 PDF는 텍스트 추출이 되지 않습니다.

## 검색 우선순위

| 순위 | 기준 |
|------|------|
| 1 | 파일명 일치 |
| 2 | 문서 내용 — 정확한 구문 일치 |
| 3 | 파일명 일부 일치 |
| 4 | 문서 내용 — 단어별 일치 |
| 5 | 문서 내용 — 단어 중 하나라도 포함 |

## 지원 파일 형식

| 형식 | 확장자 |
|------|--------|
| 한글 | `.hwp` `.hwpx` |
| PDF | `.pdf` |
| Word | `.docx` |
| Excel | `.xlsx` `.xls` |
| 텍스트 | `.txt` `.md` |

> 스캔(이미지) PDF는 텍스트 인식 불가

## 개발자용 — 소스 빌드

### 요구사항
- Node.js 18 이상
- Windows 10 / 11

### 실행

```bash
npm install
npm run rebuild   # sqlite3 네이티브 모듈 빌드
npm start
```

### 배포용 exe 빌드

```bash
npm run build
```

`dist/win-unpacked/What's in my PC.exe` 가 생성됩니다.

## 기술 스택
